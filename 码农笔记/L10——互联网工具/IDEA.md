# Mac快捷键

复制当前行 `cmd + d`
当前行上移下移 `cmd + shift + 向下箭头`
返回上一次光标所在位置 `mcd + alt + ←  或者 cmd + [`

跳出括号引号
**跳出双引号：`shift+"`  
跳出单引号：`shift+'`  
跳出括号：`shift+)`  
跳出中括号：`shift+]`**


大小写切换：`cmd + shift + u`
